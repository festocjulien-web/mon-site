I. Introduction to OSINT
1) Search Engines
Par exemple, sur Google, si vous souhaitez rechercher le contenu d'un site web spécifique, vous pouvez utiliser le mot-clé site:.
Ainsi, si vous souhaitez obtenir uniquement les résultats provenant de pastebin.com, vous pouvez inclure la requête : site:pastebin.com

Dans votre recherche Google, vous pouvez également préciser ces termes grâce à la recherche avancée de Google.

Pour d'autres exemples similaires, vous pouvez consulter la base de données de piratage de Google .
Notez toutefois que les exemples qui s'y trouvent sont davantage axés sur la recherche de sites web vulnérables et autres problèmes de ce type. Vous n'aurez pas besoin d'appliquer ces techniques dans ce module.
2) Phishing Email
Utiliser la recherche Google.
3) Creative Approach
À ce stade, vous pourriez tenter d'utiliser directement l'adresse e-mail trouvée pour interroger différentes bases de données.

Vous pouvez également essayer de deviner les noms d'utilisateur utilisés par le propriétaire de cette adresse et rechercher d'éventuels comptes en ligne sur d'autres plateformes.

De même, vous pourriez essayer de deviner d'autres adresses e-mail utilisées par cette personne et effectuer des recherches sur Internet à leur sujet.
4) Find Further Information
La recherche de noms ou de noms d'utilisateur est généralement plus efficace avec des outils personnalisés. Ces outils interrogent directement les différentes sources et ne dépendent pas des moteurs de recherche qui, pour diverses raisons, n'ont peut-être pas indexé les informations pertinentes.

Cette approche est évidemment beaucoup plus rapide que de parcourir manuellement chaque site web.

Pour les noms d'utilisateur, par exemple, un site comme usersearch.org est très utile. Si, grâce à ces outils, vous constatez qu'un site de réseau social peut contenir des informations complémentaires liées à votre recherche, vous pouvez alors les examiner directement.

Remarque : Sachez que ces sites peuvent donner des résultats pertinents, mais aussi de faux positifs.

Il existe également des sites web conçus spécifiquement pour la recherche de personnes, tels que pipl.com ou spokeo.com. Cependant, vous n'aurez pas besoin d'utiliser ces moteurs de recherche dans ce module.

De manière générale, il n'est pas nécessaire d'essayer tous les outils présentés dans ce module, et sachez que certains sont payants. Cependant, lorsque vous vous lancez seul, le site web osintframework.com constitue un bon point de départ pour trouver une liste d'outils adaptés à différents usages.
5) Explore the Website
a) Contents of the Website
Parcourez les différentes pages du site, y compris les sections moins visibles, afin de repérer des informations utiles, des noms, des adresses e-mail, des mentions légales, des fichiers PDF ou d'autres documents publiquement accessibles.

Pour la tâche suivante, gardez à l'esprit que tout le contenu intéressant n'est pas forcément affiché sur le site web. Le code source d'un site peut également contenir des informations importantes. Par exemple, vous pourriez déterminer la technologie utilisée pour sa création et trouver d'autres indices similaires permettant de trouver des sites web liés à celui que vous étudiez.

Par exemple, un site comme urlscan.io peut vous suggérer des sites similaires à celui que vous avez analysé, en fonction de leur structure.

Il est utile de savoir que pour analyser le contenu d'un site web, vous pouvez exploiter différents caches. Par exemple, Google met en cache le contenu des sites web ; ainsi, si vous trouvez un site web via une recherche Google, vous pourrez peut-être consulter une version antérieure.
b) Interesting Files
Les métadonnées d'un fichier peuvent parfois s'avérer intéressantes. Dans les images, elles sont stockées au format Exif.

Divers outils en ligne permettent d'analyser les données Exif (par exemple, jimpl.com). En ligne de commande, vous pouvez utiliser l'outil exiftool.

Voici un exemple d'extraction des coordonnées GPS d'une image avec exiftool :
exiftool -gpsposition# <fichier_image>
