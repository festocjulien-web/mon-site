I. Introduction to OSINT
▶ 1) Search Engines
Par exemple, sur Google, si vous souhaitez rechercher le contenu d'un site web spécifique, vous pouvez utiliser le mot-clé site:.

Ainsi, pour obtenir uniquement les résultats provenant de pastebin.com : site:pastebin.com

Vous pouvez aussi préciser la recherche dans Google via les filtres avancés.

Consultez aussi : La Google Hacking Database .
▶ 2) Phishing Email
Utiliser la recherche Google.
▶ 3) Creative Approach
Vous pouvez essayer :
- D’utiliser l’adresse email trouvée pour interroger des bases de données
- De deviner d’autres usernames
- De chercher si l’utilisateur existe sur d’autres plateformes
▶ 4) Find Further Information
Les outils personnalisés sont souvent plus efficaces que Google.

Par exemple : usersearch.org

Sites spécialisés : pipl.com — spokeo.com

Framework OSINT complet : osintframework.com
▶ 5) Explore the Website
▶ a) Contents of the Website
Pensez à inspecter le code source, les pages cachées, les PDFs, et utilisez des services comme : urlscan.io
▶ b) Interesting Files
Les métadonnées EXIF peuvent révéler :
- GPS
- Appareil utilisé
- Logiciels utilisés

Exemple : exiftool -gpsposition# image.jpg
